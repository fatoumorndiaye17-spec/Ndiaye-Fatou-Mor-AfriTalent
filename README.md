# AfriTalent

AfriTalent est une plateforme web qui connecte les freelances africains aux entreprises du monde entier.

## Objectif du projet
Permettre aux freelances de trouver des missions et aux entreprises de recruter facilement des talents qualifiés.

## Technologies utilisées
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Pages du site
- Accueil
- Freelances
- Tarifs
- À propos
- Contact
- Blog

## Fonctionnalités
- Design responsive (mobile, tablette, ordinateur)
- Filtrage des freelances
- Formulaire de contact
- Système de navigation simple
- Interface moderne avec Bootstrap
